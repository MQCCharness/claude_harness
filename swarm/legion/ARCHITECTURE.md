## 层级结构图

```
军团统帅（+1，主对话）—— 全局编排、冻结、Gate 门禁、回声校验
│
├── [游戏制作人] EX — 整体游戏设计文档（GDD）、技术架构决策、跨部门协调纲要
│   │
│   ├── [3D 引擎架构师] OP → 带蜂群小队
│   │   └── Three.js 核心场景、渲染管线、光影系统、相机控制、性能优化
│   │
│   ├── [游戏逻辑工程师] OP → 带蜂群小队
│   │   └── 坦克控制、敌方 AI、子弹系统、碰撞检测、关卡加载、游戏循环
│   │
│   ├── [特效工程师] OP → 带蜂群小队
│   │   └── 粒子系统、爆炸特效（雷电战机级）、弹道轨迹、环境破坏效果

```

## 信息流向

- 军令状 → 游戏制作人(EX) → 4 个 OP（各带蜂群）
- 各 OP 产出 → 军团统帅集成 → 回声校验 → 交付
- 跨 OP 依赖：特效工程师需要引擎架构师的场景接口；游戏逻辑需要引擎的碰撞接口；前端集成需要所有模块的导出接口
