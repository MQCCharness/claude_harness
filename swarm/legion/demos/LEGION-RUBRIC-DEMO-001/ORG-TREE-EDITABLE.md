# 可编辑组织架构树（草稿）

> **用法**：复制下面代码块到你的笔记或 PR 描述里改名/增删，把结果回复给统帅；统帅将同步到 `ARCHITECTURE.md` 并 bump `FREEZE_VERSION`。

```
军团统帅（主对话 / +1）
├── [EX-平台与 CLI 契约] — 冻结对外行为边界、兼容策略、与仓库治理对齐
│   └── 下级：MG-文档产物线
├── [MG-文档产物线] — HTML 单文件产物规范、验收口径、跨叶 OP 协调
│   ├── [叶 OP-主题与可访问性] — 暗色主题、对比度、默认与降级 → 挂载蜂群 4+1
│   └── [叶 OP-页脚与元数据] — 可配置版权行、转义与安全、CLI 配置面 → 挂载蜂群 4+1
```

**你可改名的占位标签**（示例）：

- `EX-平台与 CLI 契约` → 如 `EX-DevEx` / `EX-内部工具委员会`
- `MG-文档产物线` → 如 `MG-TechComm`
- `叶 OP-主题与可访问性` → 如 `OP-UX-A11y`
- `叶 OP-页脚与元数据` → 如 `OP-Content-Safety`
