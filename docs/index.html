三级可缩放治理

从轻量到重型，按需激活

根据任务风险和规模逐级升级治理强度。L1 日常小任务，L2 实质性代码改动，L3 大型多模块项目。

L1

单人直执行

统帅直接执行任务，不启动多角色。适用于简单修改、信息查询、低风险操作。

L2

蜂群模式 · 4+1 对抗

R1 需求撕裂 · R2 技术调研 · R3 核心开发 · R4a/R4b 独立测试 · +1 统帅。五段对抗式开发，写码回合强制门禁。

L3

军团模式 · EX/MG/OP

高管层战略决策 + 中层管理协调 + 末端执行带蜂群小队。六阶段强制门禁，末端叶 OP 强制满编蜂群。

L2 · 蜂群模式

4+1 对抗式开发

五个固定角色 + 统帅，通过强制多角色隔离审查，将认知盲区系统性暴露在代码落地之前。

L3 · 军团模式

EX · MG · OP 三层组织 + 六阶段门禁

军团模式是最高治理层级，为大型多模块项目引入可定制的组织架构和严格的阶段化流程。

L3-EX

高管层

战略决策 · 任务拆解 · 跨域协调 · 风险识别。默认不带蜂群。

L3-MG

中层管理

承上启下 · 多角色协调 · 汇流整合 · 质量标准。默认不带蜂群。

L3-OP

末端执行

具体执行 + 带蜂群小队。叶 OP 强制 R1–R4b 五段满编 + 时序闸门。
