# HajimiDog 开发技术栈指南

## 1. 嵌入式开发 (HajimiDog 固件)
本项目基于 STM32 标准外设库的 **裸机开发 (Bare Metal)** 实现（无 RTOS）。

### 核心编程语言与架构 (STM32F103)
*   **嵌入式 C 语言**: 位操作、指针与结构体、`volatile` 关键字。
*   **ARM Cortex-M3 内核**: NVIC 优先级管理。
*   **时钟系统 (RCC)**: 72MHz 系统时钟。
*   **通信协议**: I2C (OLED 显示屏), UART (蓝牙模块)。
*   **定时器 (TIM)**: PWM 生成 (50Hz 控制舵机), 定时中断 (非阻塞逻辑)。

### 算法与控制逻辑
*   **有限状态机 (FSM)**: 通过 `Action_Mode` 变量管理机器人行为状态。
*   **基础运动学**: 舵机角度 (0-180°) 映射为物理动作，协调相位差生成步态。
*   **非阻塞延时**: 避免阻塞 `while` 循环，实现多任务处理。

## 2. Web 开发 (可视化上位机 & 项目介绍)

### 核心技术
*   **前端框架**: React 18+, TypeScript, Vite。
*   **状态管理**: Zustand (用于跨组件共享 OLED 元素和逻辑节点状态)。
*   **节点图引擎**: `@xyflow/react` (React Flow) 用于逻辑蓝图连线。
*   **C 代码生成引擎**: 自定义 AST (抽象语法树) 解析与字符串模板拼接，将前端节点连线关系编译为符合 STM32 裸机规范的 C 语言 `while(1)` 循环与状态机代码。
*   **拖拽与缩放**: `react-rnd` 或自定义拖拽逻辑用于 OLED 元素编辑。
*   **高级 CSS3**: CSS 变量, Flexbox/Grid, 关键帧动画。

### SVG 矢量图形 (手写代码)
*   **SVG 语法**: `viewBox`, `<path>`, `<rect>`, `<circle>`, `<line>`。
*   **绘制逻辑**: 手动编写代码绘制图表（如波形图、接线图）。
