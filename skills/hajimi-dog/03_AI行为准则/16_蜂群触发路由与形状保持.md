## 1. 分层触发（提高命中率 ≠ 提高 Agent 子代理成本）

| 层级 | 含义 | 统帅动作 |
|------|------|----------|
| **L0 总线** | 每条消息 | 解析→分发→汇流→续聊；**未必**开 Agent 子代理 |
| **L1 话术命中** | 用户用到下文「触发词」 | 首段可一句点明「按蜂群总线」；**仍按**最小充分角色集决定是否 Agent 子代理 |
| **L2 强制 Agent 子代理** | 显式多角色、**满编蜂群**（五段口令）、写码回合、R2 命中等 | 必须走 Sub-agent（或诚实边界分节）；**满编军团**≠本条 |
| **L3 加码** | 高风险、集成碰壁 | 见 CLAUDE.md 高风险与集成专节 + 技能 `11` `13` |

**禁止**：用「看起来复杂」擅自 **满编蜂群**；用「提高触发率」把轻量追问一律打成五段。孤立「满编」须先澄清：蜂群五段 vs 军团架构满编（见 CLAUDE.md）。
**备忘**：用户问 **「多少轮 / 技能对抗做多久」** **≠** L2 自动满编蜂群；先区分是 **运行时满编蜂群** 还是 **技能文档侧推演深度**，再按下表 Read **`02`** 对应章节。

**勿与「留痕三档」混淆**：本表 **L0–L3** 仅表示**触发与成本分层**；CLAUDE.md 中 **叙事稿 / 结构化全记录 / 字节级镜像** 是**留痕与审计**分层，二者**不得**互借代号。

## 3. 极端情况（统帅须显式处理，防「变形」）

1. **用户新一句与已冻结 A 级事实矛盾**：指出矛盾 → 请用户确认推翻项 → **重冻结**；禁止静默以新句为准。
2. **用户从「闲聊」切回「继续改代码」**：一句恢复任务链指针（目标/待决/上次裁决）。
3. **长对话多轮**：优先 **落盘或指针**；新 Agent 子代理 **条目化摘要**，禁默认「你懂的」。
4. **「随便改改」但触及安全/持久化/规则语义**：按 **高风险** 或 **写码回合** 升格，勿当小改跳过 R3+R4。
5. **子代理结论与仓库或终端冲突**：以 **仓库与命令输出** 为准，**重发 Agent 子代理**（CLAUDE.md 已有，此处强调防漂）。
6. **豁免口语模糊**（「别太正式」「随便说说」）：若无明示豁免关键词，**不**自动退出蜂群；若用户意图是豁免，可 **一行澄清** 是否适用退出/免 Agent 子代理。
7. **同会话多主题**：若用户在同一父会话中切换**明显不同**的冻结目标（如从方案文档跳到无关脚本），统帅应**新开父会话**或启用**不同 CHAIN** 并在 `.claude/swarm/` 轮次表留痕，避免子代理目录「一桶杂烩」却假装同一形状保持。
