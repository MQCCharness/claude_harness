# Agent 技能体系全局索引与关联图谱 (Skills Master Index & Graph)

为降低上下文负担和提高检索效率，本仓库将 Skills 模块化拆分为**二级树状结构**，并建立**动态权重图谱 (Dynamic Weight Graph)**。目录名 `hajimi-dog` 为历史路径；Fork 到通用仓库时可整目录重命名，见 `docs/claude-governance/PORTING.zh-CN.md`。
**使用指南**：根据当前任务的具体需求，使用 `Read` 工具读取主 Skill，并根据【🔗技能链条】的热度权重（如 `[热度权重: 10]`）决定是否一并读取关联 Skill，**绝对不要一次性读取所有文件**。
**动态更新规则**：每次成功调用某技能链条解决问题后，必须将对应节点的权重 `+1`。

## 🛠️ 自动化流水线 (CI/CD Linter)
*   **技能体系健康检查脚本**: `.claude/skills/hajimi-dog/skills_linter.py`
    *   *说明*: 每次修改本目录下的任何 `.md` 技能文件后，必须使用 `Shell` 工具运行 `python skills_linter.py` 确保无死链且符合规范。
