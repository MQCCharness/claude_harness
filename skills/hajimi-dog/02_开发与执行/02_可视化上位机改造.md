# HajimiDog 可视化上位机改造指南

## 1. 扩展 OLED 编辑器元素 (修改 `OLEDEditor`)
*   **流程**: 在 `store/index.ts` 的 `OLEDElement` 类型中增加新类型 (如 `rect`, `circle`) -> 在编辑器画布中增加对应的渲染逻辑 -> 在 `generateOLEDCode` 中添加对应的 C 代码生成规则 (如 `OLED_DrawRect`)。

## 2. 新增逻辑节点类型 (修改 `LogicEditor`)
*   **流程**: 在 `store/index.ts` 中定义新的 `actionType` 或 `triggerType` -> 在 `components/logic/` 下创建自定义 Node 组件 -> 将新节点注册到 React Flow 实例中 -> 更新 `generateMainCode` 的 AST 解析与代码拼接逻辑。
*   **安全与校验 (Validation)**: 在 `generateMainCode` 生成 C 代码之前，**必须对节点连线和参数进行合法性校验**（例如：检查触发器是否孤立、参数是否越界）。如果校验失败，应在 UI 抛出错误提示，**阻止生成无效的 C 代码**，防止刷入单片机后导致死机。
