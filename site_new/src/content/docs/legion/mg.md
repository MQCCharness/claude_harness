# MG · 中层管理

**L3-MG（Management）** 是军团的协调枢纽，承接 EX 的战略拆解并分发给 OP 执行。

---

## 职责范围

- **承上启下**：将 EX 的战略目标翻译为 OP 可执行的具体任务
- **多角色协调**：协调多个 OP 之间的依赖和工作顺序
- **汇流整合**：汇总 OP 产出，消除冲突，打包交付给 EX
- **质量标准**：确保各 OP 产出符合统一的验收标准

---

## 执行方式

每个 MG 节点在阶段五须**至少一次独立 Agent 子代理**（`subagent_type: "general-purpose"`），禁止主对话分饰。

---

## 产出物

- **MG-PLAN**：执行计划（含 OP 拆分、时间线、依赖图）
- **MG-ROLLUP**：OP 产出汇总报告
- **MG-QUALITY-GATE**：质量门禁检查结果

---

## 默认不带蜂群

MG 层默认不配备蜂群小队。其职责是协调管理，不做具体代码实现。

---

## 典型 MG 类型

| 类型 | 示例 |
|------|------|
| **工程管理** | MG-ENGINEERING：协调前端/后端/数据库 OP |
| **体验管理** | MG-UX：协调设计/文案/可用性 OP |
| **内容管理** | MG-CONTENT：协调文档/示例/教程 OP |
| **质量保障** | MG-QA：协调测试/安全审查 OP |
