# 军团模式 (L3 · Legion)

军团模式是 Swarm/Legion 的**最高治理层级**，适用于需要多模块协调的大型项目。它引入 EX/MG/OP 三层组织架构和六阶段强制门禁。

---

## 激活条件（任一满足）

- 用户显式：`军团` / `军团模式` / `L3` / `Legion` / `多部门` / `组织架构` / `兵团` / `EX·MG·OP`
- 统帅规模裁量为 **L 级**
- 注意：仅 "军团架构满编" ≠ 蜂群五段；仅 "满编" 二字须追问澄清

## 军团 vs 蜂群

| 方面 | 蜂群 L2 | 军团 L3 |
|------|--------|--------|
| 组织 | 固定 5 角色 + 1 统帅 | 可定制 EX/MG/OP 树 |
| 流程 | 时序闸门（单轮） | 六阶段 + 多波次 |
| 规模 | 单任务 | 多模块项目 |
| 末端 | 4+1 | 叶 OP 带满编蜂群 |
| 军令状 | ②③⑤ 穿透 | 五要素全量 |
| 架构 | 无 | 用户确认 + 冻结 |

---

## Sub-agent 全强制

冻结架构中**每一个** EX/MG/OP 节点在阶段五须各至少一次独立 Agent 子代理。叶 OP 须对 R1/R2/R3/R4a/R4b 分别派发 Agent 子代理。

---

## 退出

说 `退出军团` / `不用 L3` / `降回蜂群` 即可退出 L3。
