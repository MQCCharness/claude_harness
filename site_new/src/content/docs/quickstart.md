# 快速开始

## 模式激活

### 蜂群模式 (L2)

在对话中输入以下任一命令：

| 命令 | 效果 |
|------|------|
| `/swarm` | 激活蜂群 L2 流程（R1+R2 关门 → R3 → R4） |
| `/swarm-full` | 满编蜂群五段（R1–R4b 各至少一次 Agent 子代理） |

**自动触发条件**：
- 对仓库做实质性代码改动（写码回合）
- 高风险回合（安全、鉴权、数据迁移等）
- 连续碰壁 3 轮无解

### 军团模式 (L3)

| 命令 | 效果 |
|------|------|
| `/legion` | 激活军团 L3（六阶段 + EX/MG/OP） |

---

## 写码回合最低要求

每次写码回合（实质性代码改动）强制：

1. **R1 + R2** 并行派发 → 两者关门
2. **R3** 实现（R1+R2 关门后方可启动）
3. **R4a** 可与 R3 并行（黑盒、契约级，不喂实现细节）
4. **R3 关门 + 统帅实测证据回灌 → R4b** 关门核验
5. **统帅按 R4a 表亲测** → 裁决 PASS / COND_PASS / FAIL

---

## 退出模式

| 命令 | 效果 |
|------|------|
| `退出蜂群` / `不用对抗模式` | 退出 L2 蜂群 |
| `退出军团` / `不用 L3` | 退出 L3 军团 |

---

## 常用豁免

| 命令 | 效果 |
|------|------|
| `免子代理` | 豁免 Agent 隔离（首段声明风险） |
| `跳过 R3R4` | 豁免写码回合 R3+R4a+R4b |
| `不跑实测` | 豁免 R4 实测裁决 |
| `统帅可落地代码` | 主帅直接改仓 |
