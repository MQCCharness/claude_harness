# 豁免口令速查

以下是 Swarm/Legion 体系中所有可用的豁免口令。

---

## 蜂群豁免

| 口令 | 效果 |
|------|------|
| `退出蜂群` | 退出 L2 蜂群模式 |
| `不用对抗模式` | 退出 L2 蜂群模式 |
| `免子代理` | 豁免 Agent 隔离（统帅首段须声明风险） |
| `不要 Task` | 同上，豁免 Agent 隔离 |
| `跳过 R3R4` | 豁免写码回合 R3+R4a+R4b |
| `免 R3R4` | 同上 |
| `不跑实测` | 豁免 R4 实测裁决 |
| `豁免验收门禁` | 同上 |
| `免执行配置闸门` | 跳过 Gate 0 (EXEC_CFG_GATE) |
| `模型主帅定` | 同上 |

---

## 军团豁免

| 口令 | 效果 |
|------|------|
| `退出军团` | 退出 L3 军团模式 |
| `不用 L3` | 同上 |
| `降回蜂群` | 退到 L2 蜂群 |
| `豁免军团阶段` | 压缩军团阶段（须明示范围） |
| `军团阶段压缩` | 同上 |
| `轻量军团` | 仅减落盘，不减阶段二～三 |
| `LEGION_LITE` | 同上 |
| `消费型军团` | 同上 |
| `免角色落盘` | 豁免 runs/ 盘稿 |

---

## 统帅权限

| 口令 | 效果 |
|------|------|
| `统帅可落地代码` | 主帅直接改仓（限治理白名单） |
| `主帅直接改仓` | 同上 |
| `主帅改业务仓` | 扩展主帅直接改仓到业务树 |

---

## 重要说明

1. 豁免口令**必须用户明示**，不得由统帅推断或默认应用
2. 豁免后须在首段声明风险（如适用）
3. 部分豁免不可叠加（如 `退出蜂群` 后 `/swarm` 会重新激活）
